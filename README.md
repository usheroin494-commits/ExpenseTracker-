# Expense Tracker - Трекер личных расходов

Валиханов Ахмадали Мухаммадалиевич 

## Описание программы
Expense Tracker — это графическое приложение для учёта личных финансов. Позволяет добавлять расходы с указанием суммы, категории и даты, фильтровать записи по категории и дате, а также подсчитывать общую сумму расходов за выбранный период. Данные автоматически сохраняются в JSON-файл.

## Установка и запуск
1. Убедитесь, что установлен Python 3.6 или выше.
2. Скачайте файл `expense_tracker.py`.
3. Запустите из терминала:
   ```bash
